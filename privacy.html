Jiro Bot Privacy Policy

How your data is collected, used, and protected

Effective: April 16, 2026

1. Overview

This Privacy Policy explains how Jiro Bot collects and processes limited data to provide functionality, improve performance, and prevent abuse.

2. Data We Collect

We may collect:

  • Discord User IDs
  • Discord Server IDs
  • Command usage logs
  • Server configuration settings

No private messages or sensitive personal data are intentionally collected.

3. How We Use Data

  • To run bot features correctly
  • To store server preferences
  • To improve performance and reliability
  • To detect abuse or spam behavior

4. Data Storage

Data may be stored securely using third-party databases such as Supabase. Data is only kept as long as necessary for bot functionality.

5. Data Sharing

We do not sell or trade data. Data is only shared if required by law or to protect system integrity.

6. Third-Party Services

Jiro Bot depends on Discord APIs and external hosting providers. Their policies also apply.

7. Security

We take reasonable steps to protect data, but no system is 100% secure.

8. User Rights

You may request deletion or clarification of stored data by contacting support.

9. Retention

Data is retained only as long as needed. Inactive data may be removed periodically.
